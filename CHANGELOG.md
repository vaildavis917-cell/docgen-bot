# Changelog

Все изменения в проекте документируются в этом файле.

## [2.0.1] - 2025-01-09

### Добавлено
- **Генерация адресов по всему миру** - 45+ стран:
  - Европа: Франция, Италия, Испания, Нидерланды, Швеция, Швейцария, Австрия, Бельгия, Португалия, Норвегия, Дания, Финляндия, Чехия
  - Азия: Япония, Китай, Южная Корея, Индия, Таиланд, Малайзия, Филиппины, Индонезия, Вьетнам, ОАЭ, Сингапур, Израиль, Турция
  - Америка: Канада, Мексика, Бразилия, Аргентина, Чили, Колумбия
  - Океания: Австралия, Новая Зеландия
  - Африка: ЮАР, Египет, Нигерия, Кения

- **Генерация карт всего мира** - 14 платёжных систем:
  - Международные: Visa, Mastercard, American Express, Discover, Diners Club
  - Азия: UnionPay (Китай), JCB (Япония), RuPay (Индия), Troy (Турция)
  - Европа: Maestro, MIR (Россия)
  - Латинская Америка: Elo, Hipercard (Бразилия)
  - Африка: Verve (Нигерия)

- **Выбор формата видео** при уникализации (.mp4, .mov, .avi, .mkv)
- **Выбор количества вариаций** (1-10 копий) для фото и видео

### Оптимизации
- **Ограничение 720p** - видео автоматически уменьшается до 720p для ускорения
- **Аппаратное ускорение** - hwaccel auto для GPU
- **Отключён шум** - убран медленный фильтр noise
- **Случайные метаданные** - при каждой уникализации меняются:
  - Дата создания
  - Энкодер
  - Заголовок
  - Комментарий
- **Увеличен таймаут** до 10 минут для больших файлов

### Исправлено
- Долгая обработка видео (было ~2 мин, стало ~20-30 сек)

## [2.0.0] - 2025-01-08

### Добавлено
- **Админ-панель** - полноценная панель управления для администратора
  - Управление VIP пользователями (добавление/удаление)
  - Бан/разбан пользователей
  - Просмотр статистики бота
  - Рассылка сообщений всем пользователям
  - Режим технических работ (Maintenance Mode)
  - Информация о пользователях по ID
  - **Настройки антифлуда** через админ-панель

- **Система безопасности**
  - AntiFlood защита с настраиваемыми параметрами
  - Rate Limiting (30 запросов/минуту)
  - Input Validation для всех входных данных
  - Security Logging для подозрительной активности
  - Bot Detector для защиты от автоматизации

- **Пересылка медиа** - все фото и видео пересылаются на указанный ID
- **Функция "Сообщить об ошибке"** - пользователи могут отправлять баг-репорты
- **HTTP/2 поддержка** - улучшенная производительность сети

### Оптимизации
- Асинхронная обработка видео и фото
- Connection Pooling (100 макс соединений, 50 keep-alive)
- Кэширование для часто используемых данных
- Оптимизация для работы с 300+ пользователями

### Исправлено
- NameError для модуля 'os' в функции start()
- Рекурсия в safe_edit_text()
- Отсутствующий аргумент в download_website()
- Хардкод путей заменён на относительные через __file__
- Все 80+ кнопок теперь имеют обработчики

### Известные проблемы
- При работе двух экземпляров бота возникает 409 Conflict
- Возможно задвоение сообщений при 409 Conflict (см. KNOWN_ISSUES.md)
