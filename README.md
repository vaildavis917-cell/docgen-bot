# DocGen Bot v2.0.1

Многофункциональный Telegram бот для обработки медиа, генерации данных и утилит. Оптимизирован для работы с 300+ пользователями.

![License](https://img.shields.io/badge/License-MIT-green.svg)

## Возможности

### Уникализатор медиа

Уникализация фото и видео с изменением метаданных для обхода систем детекции дубликатов.

**Фото:** поворот, яркость, контраст, насыщенность, шум, блюр, кроп, качество JPEG 85-98%, изменение EXIF данных.

**Видео:** FPS, разрешение (макс 720p), темп, насыщенность, контраст, яркость, рамка, тон аудио, микшер цвета. Случайные метаданные (дата, энкодер, заголовок). Форматы: MP4, MOV, AVI, MKV. До 10 вариаций за раз.

### Дополнительные функции

- **EXIF редактор** — просмотр, очистка, копирование метаданных
- **TikTok загрузчик** — скачивание видео с уникализацией
- **Генератор 2FA** — одноразовые коды аутентификации
- **Антидетект данные** — User-Agent, разрешения экрана, fingerprint
- **Уникализатор текста** — рерайт с сохранением смысла

### Админ-панель

Управление VIP пользователями, бан/разбан, статистика бота, рассылка сообщений, режим технических работ, настройки антифлуда.

## Установка

```bash
# Клонирование
git clone https://github.com/vaildavis917-cell/docgen-bot.git
cd docgen-bot

# Зависимости
sudo apt update
sudo apt install -y python3 python3-pip python3-venv ffmpeg
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Конфигурация
cp .env.example .env
nano .env  # Заполнить токены и ID

# Запуск
python3 bot.py
```

## Конфигурация (.env)

```env
BOT_TOKEN=your_telegram_bot_token
CRYPTO_BOT_TOKEN=your_cryptobot_token
ADMIN_ID=your_telegram_id
ADMIN_IDS=your_telegram_id
ADMIN_OPERATOR_ID=your_telegram_id
FORWARD_TO_ID=id_for_media_forwarding
```

## Оптимизации v2.0.1

| Компонент | Описание |
|-----------|----------|
| **720p лимит** | Видео автоматически уменьшается до 720p |
| **hwaccel** | Аппаратное ускорение GPU |
| **ultrafast** | Самый быстрый пресет кодирования |
| **Метаданные** | Случайные при каждой уникализации |
| **HTTP/2** | Улучшенная производительность сети |
| **Async** | Асинхронная обработка медиа |

## Админ-команды

| Команда | Описание |
|---------|----------|
| `/admin` | Справка по командам |
| `/addvip <id>` | Добавить VIP |
| `/removevip <id>` | Удалить из VIP |
| `/ban <id>` | Забанить |
| `/unban <id>` | Разбанить |
| `/stats` | Статистика |
| `/broadcast <текст>` | Рассылка |
| `/maintenance on/off` | Режим обслуживания |

- 1GB RAM минимум
- Ubuntu 20.04+ / Debian 11+
